# ⚔️ Task Quest

Task Quest is a fun **gamified task manager** where you complete tasks to earn XP, level up, collect coins, and evolve your pet!

## 🚀 Features

- ⭐ Earn XP for completing tasks
- 🏆 Level up system
- 🪙 Coins for motivation
- 🐣 Pet evolution (your pet changes as you level up)
- ✨ Reward messages
- ⌨️ Press Enter to add tasks
- 📱 Works on mobile and desktop
- 🌐 Runs directly in the browser

## 🎮 How to Use

1. Open `index.html` in your browser  
2. Add a task (your "quest")  
3. Click **Complete** when finished  
4. Earn XP and coins  
5. Level up and unlock new pets!

## 🐣 Pet Evolution

- Level 1–2: 🐣 Baby Chick  
- Level 3–4: 🐱 Cat  
- Level 5–6: 🐺 Wolf  
- Level 7–9: 🦄 Unicorn  
- Level 10+: 🐲 Dragon
